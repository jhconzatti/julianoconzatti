## About This Project

This repository is the foundation of my personal brand website and professional portfolio.
It was designed to communicate executive-level technical positioning with speed, clarity, and scalability.

The platform is multilingual by design, supporting:

- Portuguese (PT-BR)
- English (EN-US)
- Spanish (ES)

---

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Sanity (Headless CMS for the blog)

---

## Key Features

- Native i18n architecture using context plus centralized translation structures
- Design system-oriented UI with Tailwind CSS and shadcn/ui components
- Headless CMS integration with Sanity for scalable blog content management
- Performance-focused frontend foundation for fast loading and responsive user experience

---

## Getting Started

Prerequisites:

- Node.js
- npm

Install dependencies:

	npm i

Start the local development server:

	npm run dev

Build for production:

	npm run build
